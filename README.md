# social-media-fake-account-and-toxic-comment-detector
AI-powered Social Media Safety System that detects fake accounts and toxic comments using 7 Machine Learning models (Logistic Regression, Decision Tree, Random Forest, XGBoost, LightGBM, SVM, Stacking Ensemble) achieving 98% accuracy, with BERT NLP, SMOTE balancing, Neural Networks, and live Streamlit web app.
